Changelog

Version 1.5.2 - 12/10/2016

  • Created a new window named My Checkbook Ledgers. This consolidates the Open and Ledger Manager windows. All functions operate the same as before.
  • Updated confirmation message when deleting a ledger in My Checkbook Ledgers (formally known as Ledger Manager). The message now provides a clear list of items that need to be restored.
  • A new directory was created in My Checkbook Ledgers in the My Documents directory named Settings. All ledgers now have their own settings file. All customization settings available in Options are now unique to each ledger.
  • Customize Toolbar is now unique to each ledger. If multiple people are using Checkbook to manage their accounts they can set it up the way they want.
  • Advanced Filter now loads your Categories and Payees when the window is activated. If you edited your Categories or Payees while Advanced Filter was open you will not have to close and reopen to load them anymore.
  • Alt key shortcuts have been added to all menu items.
  • Added Duplicate Transaction(s) function. Allows you to duplicate the currently selected transactions.

Version 1.5.1 - 11/20/2016

  • Minor bug fixes.
  • Updated confirmation message when deleting a ledger in Ledger Manager. When restoring from the recycle bin, if budgets have been created the budgets file ".bgt" must also be restored.

Version 1.5.0 - 9/18/2016

  • Major bug fixes for Version 1.4.9.

Version 1.4.9 - 9/18/2016

  • Major bug fixes for Version 1.4.8.

Version 1.4.8 - 9/17/2016

  • Added Advanced Filter feature in the Tools menu. This allows you to fine tune a filtered search based on Dates, Categories and/or Payees, Cleared or Uncleared, Receipt or No Receipt.

Version 1.4.7 - 7/18/2016

  • Added new window for printing and previewing charts. Clicking Print Chart in Spending Overview now opens a new window with a small scale preview area.
  • Added new default directory location in Options. There is now an option to save a default location to choose receipt files from when making a new transaction or editing an existing transaction.

Version 1.4.6 - 7/9/2016

  • Added new window called Spending Overview Charts which can be accessed from Spending Overview. This new window allows you to view and print your totals from the Category/Payee table in Spending Overview in a pie style chart or a donut style chart.

Version 1.4.5 - 6/30/2016

  • Added new column in Budgets called Percent Used indicating how much of your budget you have used for the current month.

Version 1.4.4 - 6/23/2016

  • Added new feature in Options for setting default directories for Save & Open What if Scenario, Import Transactions, Export Transactions, and Backup & Restore ledgers.

Version 1.4.3 - 6/15/2016

  • Fixed bug where all transactions would not load after balancing account when all transactions were cleared.

Version 1.4.2 - 6/11/2016

  • Added Export Transactions feature in the Tools menu. This allows you to export all loaded transactions to a CSV (Comma delimited) (*.csv) file.

Version 1.4.1 - 6/3/2016

  • Most Used Categories/Payees no longer shows in the taskbar when opened to be consistent with all other commands.
  • Added help button in Most Used Categories/Payees.

Version 1.4 - 5/31/2016

  • Added Most Used Categories/Payees feature in View. This allows you to see how many occurrences there are for each category and payee in your ledger. You can calculate based on year or your entire ledger. Your total payments and deposits are also calculated and are sortable.

Version 1.3 - 5/22/2016

  • Made Checkbook public under the GNU GENERAL PUBLIC LICENSE.

Version 1.2 - 5/22/2016

  • The Starting Balance and Cleared Balance values in Account Details are no longer highlighted. This was not necessary.
  • Moved the Overall Balance value in Account Details next to Ledger Status to keep the highlighted values together.

Version 1.1 - 5/16/2016

  • Added Customize Toolbar feature in Options. Allows you to add, remove, and sort button icons on the toolbar. This is a per Windows User setting.
  • Added Budgets feature in View. Allows you to keep track of how much you are spending on certain categories for the current month and set a spending limit.

Version 1.0 - Initial Release - 4/20/2016
