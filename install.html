Checkbook is not digitally signed. This means that Windows and web browsers will warn you that it may be harmful. But why would I install a program that Windows doesn't recognize as safe? Good question. Well, digital signatures or certificates are only a means of providing the user with a proof that the author is trust worthly. These security scans don't actually know if the code in the application is safe or not. So if you trust the author then you are good to go. Checkbook is intended to be a free application and digital certificates are quite expensive, too expensive for a free application.

Below are some of the warnings you may be presented with while trying to download and install Checkbook on your computer.

  • Your virus protection software could potentially flag it as a threat.
  • Windows SmartScreen: Click "More Info", then "Run Anyway" to proceed. (see below)
  • Web browsers may warn you that you may have downloaded a harmful file. (see below). The more you download a particular file your web browser will stop warning you.
