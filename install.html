Please review below before installing

Checkbook is not digitally signed. Checkbook is intended to be a free application and digital certificates are too expensive for a free application. This means that Windows and web browsers will warn you that it may be harmful.

Below are some of the warnings you may be presented with while trying to download and install Checkbook on your computer.

  • Your virus protection software could potentially flag it as a threat.
  • Windows SmartScreen: This is a check to see if the application is signed with a digital certificate. If it does not have one it will let you know it is from an "Unknown Publisher". Click "More Info", then "Run Anyway" to proceed. (see below)
  • User Account Control: This is another check which basically gives you one last warning that it is from an "Unknown Publisher". Click "Yes" to proceed.
  • Web browsers may warn you that you may have downloaded a harmful file. (see below). The more you download a particular file your web browser will recognize that you trust it and will stop warning you.
