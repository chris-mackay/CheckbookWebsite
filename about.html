About Checkbook

I created Checkbook for myself and have been developing it in my free time for the past five years. I wanted to make a simple application that did exactly what I wanted. I have developed it to a point where I want to make it available for others to use in hopes that it will be as helpful to others and it is for me. Checkbook is free, so share it and tell everyone you know. You can check for the latest version with the "Check for Update" menu item in the "Tools" menu.

Checkbook has also been made public under the GNU GENERAL PUBLIC LICENSE. If you want to contribute to Checkbook you can find the code repositories on GitHub.
