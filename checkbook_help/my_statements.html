My Statements

My Statements allows you to save your monthly bank statements for attaching to transactions. Below is a list of options available in My Statements.

  • New Statement: Allows you to add a new statement to attach to transactions. Provide a name and a file of any file type.
  • Delete Statement: Deletes the currently selected statement. This also removes all links to transactions in your ledger.
  • Rename Statement: Rename the currently selected statement. This also renames all links to transactions in your ledger.
  • View Statement: Allows you to view the currently selected statement.
