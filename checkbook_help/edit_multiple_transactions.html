Edit Multiple Transactions

If you have multiple transactions that you want to update with the same information follow the steps below.

  1. Select the transactions that you want to edit.
  2. Go to Edit >Edit Transaction(s)... > value you want to edit. You can access these items in the right click menu as well.
  3. Specify or select the new value to update.
  4. Click the Update button.
