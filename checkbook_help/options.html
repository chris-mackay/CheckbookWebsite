Options

Options allows you to personalize the appearance of how your ledger data is displayed. The options will be saved for the current ledger only, allowing you to personalize each one of your ledgers. If more than one person is using Checkbook on your computer they can personalize their own ledgers.

Ledger Graphics

Colors & Toolbar

  • Grid Color: If you choose to show the grid lines on your ledger you can change the color here.
  • Uncleared Highlight Color: If you choose to highlight uncleared transactions you can change the color here.
  • Row Selection Color: Allows you to change the highlight color of the currently selected transactions.
  • Alternating Row Color: If you choose to show coloring of alternating rows in your ledger you can change the color here.
  • Randomize: Allows you to try all different colors for the options described above. Keep randomizing until you find something you like or something to start with.
  • Restore Default Colors: Resets all the color options described above to Checkbook default colors.
  • Customize Toolbar: Allows you to add, remove, and sort button icons on the toolbar.

Grid Options

  • Cell Border: Each value in your ledger will be enclosed with a box border.
  • Row Grid Lines: The top and bottom of each row in your ledger with have a border.
  • Column Grid Lines: The left and right side of each column in your ledger will have a border.

Color Options

  • Color Uncleared: If this option is checked each transaction that is uncleared will be highlighted with the Uncleared Highlight Color.
  • Color Alternating Rows: If this option is checked every other row in your ledger will be colored with the Alternating Row Color.

Default Directories

  • Allows you to choose commonly used directories for quicker access of files.
