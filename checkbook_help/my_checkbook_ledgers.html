My Checkbook Ledgers

My Checkbook Ledgers allows you to manage all your ledgers from one place. Below is a list of options available in My Checkbook Ledgers.

  • New Ledger: Allows you to create a new ledger for keeping track of a checking account. Enter a name and starting balance for your ledger.
  • Delete Ledger: Allows you to delete a ledger. Ledgers and their associated files will be sent to the recycle bin. Below is a list of files that would need to be restored from the recycle bin if you changed your mind.
    • Checkbook Ledger (.cbk): This file contains all your transaction data. The file will be named LedgerName.cbk.
    • Checkbook Budget File (.bgt): This file contains all your budget information. If you have not created budgets for your ledger this file will not exist. The file will be named LedgerName.bgt.
    • Checkbook Settings File (.cks): This file contains all your settings from Options. The file will be named LedgerName.cks.
    • Receipts (Folder): A folder containing all your saved receipts that are attached to transactions. The folder will be named LedgerName_Receipts.
  • Rename Ledger: Allows you to rename a ledger. You can rename the currently open ledger as well.
  • Backup Ledger: Creates a backup folder containing the ledger and it's associated files. The folder will be named "LedgerName_Backup".
  • Restore Ledger: Allows you to restore a previous version of a ledger. In the folder dialog select the folder named "LedgerName_Backup". You can restore any ledger from the list at any time.
