Spending Overview

Spending Overview gives you the big picture and lets you see where every penny of your money is going and coming from each month of the year. The items in the Edit (What if Tools) menu allow you to change your totals to create hypothetical scenarios to help you with finanical planning.

Filter Options

Perform calculations on your ledger data using Filter Options. At any time you can calculate based on year, categories or payees, and payments or deposits. Keep in mind that selecting any of these options while currenly editing your totals with the What if Tools in Edit your values will be reset to your actual ledger data. These options will also be disabled during creating an Empty What if Scenario. If you want to reset your data and enable these options again select either 'Reset All Expenses' if you have payments selected or 'Reset All Incomes' if you have deposits selected.

Menu Items

File

  • Save What if Scenario: Checkbook saves your data in a folder titled "day-month-year_'Filename'_Whatif Scenario. The folder that Checkbook creates should not be renamed. If you want to organize your scenarios you may place the folder within another folder. Scenarios can be overwritten.
  • Open What if Scenario: Select the folder containing your What if Scenario and you will be able to continue editing your data.

Edit (What if Tools)

The features in Edit allow you to change your current spending habits and pretend they are something else. You can create new monthly payments, edit existing payments, and remove entire entries. You can also copy selected monthly totals to the next month, selected months, or the rest of the year. Your account details will update based on this new information. Making changes in Spending Overview will not alter your actual ledger data. These features turn Spending Overview into a sandbox for you to try different monthly scenarios. These features will only be enabled when the selected year in "Filter Options" is the most recent year in your ledger. However, data from a previous year can be used as a starting point for a What if Scenario. Select the year you want to start with and save it, then open it. You can start editing your data at any time.

  • Create Monthly Expense/Income: Create a new monthly total to add to your scenario. Select a category or payee depending on which you have selected in Filter Options and set an amount. This amount will be applied to each month of the year.
  • Edit Expenses/Incomes: Allows you to change the amount of a selected monthly total in the table. You can edit multiple amounts at once.
  • Remove Expenses/Incomes: Allows you to completely remove a monthly total from the table. You can remove multiple amounts at once.
  • Remove Categories/Payees: Allows you to completely remove a category or payee from the table. You can remove multiple categories or payees at once.
  • Copy to Next Month: Allows you to copy the selected totals in the table to the next month. You can only copy totals from one month at a time.
  • Copy to Selected Months: Allows you to copy the selected totals in the table to whichever months you choose from the selection window. You can only copy totals from one month at a time.
  • Copy to Rest of Year: Allows you to copy the selected totals in the table to the rest of the year. You can only copy totals from one month at a time.
  • Create Empty Scenario: Start with a blank slate. Creating an empty What if Scenario allows you to model an entire year from scratch so you can predict what your account balance will be. Current Year Details are set to zero and you are free to make them whatever you wish. Depending on whether you are creating an empty scenario containing payments or deposits you will be prompted asking if you would like to apply a monthly total to each month to get your calculations started. If you know how much you spend or make each month this amount can be applied right at the start.

Modeling Options

Switch between the current year and next year. When "Current Year" is selected you will be altering the outcome of what the current year will be, or what it would have been. When "Next Year" is selected your current overall balance from your ledger is used against the "Current Year Details" so you can see what next year would be if you actually acted out the "Current Year Details".

  • Current Year Details: Displays a summary of the totals for the currently selected year or the result of altering values using the What if Tools in the Edit menu.
  • Overall Account Details: Displays your Overall Balance and Ledger Status. These values work the same as the Overall Balance and Ledger Status from your ledger. These also reflect changes made while using the What if Tools in the Edit menu.

View

  • Spending Overview Charts: See your monthly totals from the Category/Payee table displayed in a color coded chart.
  • Reset All Expenses/Incomes: Allows you to reset the Category/Payee and Monthly Income table to the actual values from your ledger when you are finished editing.

Tools

  • Export Spending Overview Tables: Allows you to export the Category/Payee and Monthly Income table to a CSV (Comma delimited) (*.csv) file which can be opened with Microsoft Excel. Continue performing calculations to your heart's content.
