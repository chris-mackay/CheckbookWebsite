Want to help make Checkbook?

Checkbook is written in VB.NET using Visual Studio and targets Microsoft .NET Framework 4.5. Below are the repositories on GitHub containing all the code files necessary to edit, run, and package Checkbook.

  • Checkbook - Application
  • CheckbookMessage - Custom messagebox
  • CheckbookPackaging - Files and folder structure for packaging Checkbook with an Inno Setup script.
